## 機能
- **カレンダー同期**: 設定画面で登録したURLのカレンダーを自動同期
- **自動同期**: 設定画面で同期する時間を指定
- **手動同期**: カレンダーの手動同期(v2.0から)
- **色付け**: カレンダー名の色変更機能(v2.0から)
- **カレンダー名の設定**:カレンダー名の設定が可能(v2.0から)

## 必要な設定
### 1. Googoleカレンダーの非公開ical形式の非公開URL取得
1. Googleアカウントでログイン
2. [Googleカレンダー](https://calendar.google.com/calendar) にアクセス
3. 非公開ical形式の非公開URL取得

### 2. プラグイン設定

1. Exmentのプラグイン設定画面を開く
2.「Googleカレンダービューアー」の設定を開く
3. 設定保存
